## 프로젝트 개요
+ 도서관 수는 늘었지만 사서의 수가 감소하여 인력이 부족하고 펜데믹 이후 업무 강도는 올라가 도서관 운영의 어려움을 겪고 있다는 문제를 발견했습니다.
+ Libro 프로젝트는 도서관 사서의 반복 업무를 대신해 업무 부담을 줄이고 도서관 이용자에게는 편리를 제공합니다.
